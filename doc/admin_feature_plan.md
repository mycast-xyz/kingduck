# 어드민 페이지 기능 개선 플랜

## 현재 구현된 기능

프로젝트를 분석한 결과 다음과 같은 어드민 기능들이 이미 구현되어 있습니다:

- **게임 관리** (`AdminGameList.svelte`): 게임 목록 조회, 게임별 캐릭터/아이템/타입 수 확인
- **캐릭터 관리** (`AdminCharacterList.svelte`, `AdminCharacterContent.svelte`): 캐릭터 목록, 검색, 필터링, 상세 정보 조회
- **칸반 보드** (`AdminKanban.svelte`): 작업 관리용 칸반 시스템 (드래그 앤 드롭 지원)
- **대시보드** (`AdminDashBoard.svelte`): 통계 카드, 차트, 프로젝트 목록

## 제안하는 추가 기능

프로젝트가 **가챠 게임 정보 종합 플랫폼**임을 고려하여 다음 기능들을 제안합니다:

### 1. 캘린더/이벤트 관리

**목적**: 게임별 가챠 배너, 이벤트 일정을 관리

- **이벤트 등록/수정/삭제**
  - 이벤트명, 게임, 시작일, 종료일, 이벤트 타입 (가챠/이벤트/업데이트)
  - 이미지 업로드, 공식 링크 연결
  - 메타데이터 (JSON) 편집 기능
- **이벤트 목록 조회**
  - 게임별 필터링
  - 날짜 범위 검색
  - 이벤트 타입별 필터링
- **캘린더 미리보기**
  - 사용자 페이지에 표시될 형태 미리보기

### 2. 아이템 관리

**목적**: 게임별 아이템(무기, 유물, 성유물 등) 관리

- **아이템 목록 조회**
  - 게임별 필터링
  - 검색 기능 (이름, 타입)
  - 페이지네이션
- **아이템 등록/수정/삭제**
  - 기본 정보 (이름, 설명, 등급)
  - 이미지 업로드
  - 게임별 메타데이터
- **일괄 업로드**
  - CSV/JSON 파일을 통한 대량 등록

### 3. 사용자 관리

**목적**: 플랫폼 사용자 및 권한 관리

- **사용자 목록**
  - 가입일, 활동 현황, 권한 수준
  - 검색 및 필터링
- **권한 관리**
  - 관리자, 에디터, 일반 사용자 권한 부여
  - 특정 게임/기능에 대한 세부 권한 설정
- **활동 로그**
  - 사용자별 접속 기록, 작업 이력

### 4. 크롤러/파서 관리

**목적**: 게임 공식 업데이트 자동 수집 및 파싱 관리

- **크롤러 상태 모니터링**
  - 각 게임별 크롤러 실행 상태
  - 마지막 수집 시간, 성공/실패 여부
  - 에러 로그 확인
- **수동 실행**
  - 특정 게임의 크롤러 즉시 실행
  - 파싱 규칙 테스트
- **파싱 결과 검토**
  - 자동 수집된 데이터 검토 및 수정
  - 승인/거부 기능

### 5. 컨텐츠 승인 시스템

**목적**: 자동 수집 또는 사용자 제보 컨텐츠 검토

- **대기 중인 컨텐츠 목록**
  - 크롤러로 수집된 신규 캐릭터/이벤트
  - 사용자가 제보한 정보
- **검토 및 승인/거부**
  - 정보 정확성 확인
  - 수정 후 승인 또는 거부
  - 거부 사유 기록

### 6. 팀 빌드 추천 관리

**목적**: 캐릭터별 팀 조합 추천 관리

- **팀 조합 등록/수정**
  - 메인/서브 캐릭터 설정
  - 장비/유물 추천
  - 시너지 태그 추가
- **검증 기능**
  - 중복 조합 확인
  - 출시되지 않은 캐릭터 경고

### 7. 통계 및 분석

**목적**: 플랫폼 사용 현황 파악

- **방문자 통계**
  - 일별/월별 방문자 수
  - 페이지뷰, 평균 체류 시간
  - 인기 게임/캐릭터
- **컨텐츠 완성도**
  - 게임별 캐릭터 정보 완성률
  - 누락된 데이터 현황
- **검색 키워드 분석**
  - 사용자가 자주 검색하는 키워드
  - 검색 결과 없는 쿼리
- **시스템 모니터링** (신규 추가)
  - [상세 명세서: admin_system_monitoring.md](./admin_system_monitoring.md)
  - 서버 리소스(CPU, MEM, Disk) 및 프로세스 상태 모니터링

### 8. 공지사항/FAQ 관리

**목적**: 사용자 안내 및 고객 지원

- **공지사항 등록/수정/삭제**
  - 제목, 내용, 중요도
  - 게시 기간 설정
  - 상단 고정 기능
- **FAQ 관리**
  - 카테고리별 FAQ 등록
  - 검색어 태그 설정
  - 조회수 추적

### 9. 이미지/미디어 관리

**목적**: 캐릭터, 아이템 이미지 일괄 관리

- **미디어 라이브러리**
  - 업로드된 모든 이미지 조회
  - 미사용 이미지 확인 및 삭제
- **이미지 최적화**
  - WebP 변환 자동화
  - 리사이징 일괄 처리
- **CDN 관리**
  - 캐시 퍼지
  - URL 생성

### 10. 타입/엘리먼트 관리

**목적**: 게임별 속성, 직업, 무기 타입 등 관리

- **타입 목록 조회**
  - 게임별 필터링
  - 타입 카테고리 (속성/직업/무기 등)
- **타입 등록/수정/삭제**
  - 이름, 아이콘, 설명
  - 색상 코드 설정

### 11. 배치 작업 관리

**목적**: 대량 데이터 처리 및 정기 작업 관리

- **스케줄 작업 조회**
  - 크롤러 실행 스케줄
  - 데이터 백업 스케줄
- **수동 배치 실행**
  - 이미지 일괄 다운로드
  - DB 마이그레이션
- **작업 이력**
  - 실행 시간, 소요 시간, 결과

### 12. 설정 관리

**목적**: 시스템 전반 설정

- **일반 설정**
  - 사이트명, 로고, 테마
  - SEO 메타 태그 기본값
- **API 설정**
  - 외부 API 키 관리
  - Rate limiting 설정
- **알림 설정**
  - 이메일/슬랙 알림 설정
  - 에러 알림 임계값

8. **팀 빌드 추천 관리**

12. **설정 관리**
